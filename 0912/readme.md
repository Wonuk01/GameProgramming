### 2. 아스키 코드 (ASCII Code)
- **정의**: 문자에 대해 약속된 코드 값  
- 예시: `'a'` → 미리 정해진 코드 값으로 처리/전송됨

---

### 3. 스캔 코드 (Scan Code)
- **정의**: 각각의 키(key)에 할당된 고유 코드 값  
- 키보드 입력 자체를 구분하는 코드

---

### 4. 범위 내 난수 생성
- 특정 범위(예: `1~45`, `0~99`)의 정수를 생성하는 방법  
- `rand()` 함수와 **나머지 연산자 `%`**를 조합하여 사용
  ```c
  int num = rand() % 45 + 1; // 1 ~ 45 범위의 난수
  int num2 = rand() % 100;   // 0 ~ 99 범위의 난수

---
